deslizaJogo :: Int -> Jogo -> Jogo #

A funçao deslizaJogo faz o mapa deslizar ao longo do tempo onde a última linha do mapa é retirada e adicionada uma nova linha, gerada aleatóriamente, no topo do mapa. Utiliza a funçao estende mapa para tal. @ deslizaJogo :: Int -> Jogo -> Jogo deslizaJogo num (Jogo (Jogador (x,y)) (Mapa l ls)) = (Jogo (Jogador (x,y-90)) (estendeMapa (Mapa l (init ls)) num))
